Linux Commands Reference

Essential CLI Tools for the Network+ Exam
⚡ Priority Commands: Know ping, traceroute, netstat/ss, nslookup/dig, ssh, and tcpdump cold — these appear most in scenario-based questions.
🖧

Network Configuration & Info

CommandPurpose
ifconfigDisplay/configure network interfaces LEGACY
ip addrDisplay IP address info — modern replacement for ifconfig
ip linkShow and manage network interfaces
ip routeView and manage the routing table
routeDisplay routing table LEGACY
hostnameDisplay or set the system hostname
📡

Connectivity & Troubleshooting

CommandPurpose
pingTest reachability via ICMP EXAM CRITICAL
tracerouteTrace path to destination (Windows: tracert) EXAM CRITICAL
mtrCombines ping + traceroute in real-time
curlTransfer data from a URL — tests HTTP connectivity
wgetDownload files from the web
🔍

DNS Tools

CommandPurpose
nslookupQuery DNS records EXAM CRITICAL
digDetailed DNS lookup — more powerful than nslookup EXAM CRITICAL
hostSimple DNS lookup utility
🔌

Network Connections & Ports

CommandPurpose
netstatShow active connections, ports, routing LEGACY EXAM CRITICAL
ssModern replacement for netstat EXAM CRITICAL
nmapNetwork and port scanner
📶

Wireless

CommandPurpose
iwconfigConfigure wireless interfaces LEGACY
iwModern wireless configuration tool
nmcliNetworkManager command-line interface
📦

Packet Capture & Analysis

CommandPurpose
tcpdumpCapture packets from the command line EXAM CRITICAL
wiresharkGUI packet analyzer — also know CLI version: tshark
🔐

File Transfer & Remote Access

CommandPurpose
sshSecure remote login — port 22 EXAM CRITICAL
scpSecure file copy over SSH
sftpSecure FTP over SSH
ftpFile transfer — unencrypted, legacy LEGACY
🛡️

System & Firewall

CommandPurpose
iptablesConfigure Linux firewall rules LEGACY
nftables / nftModern firewall replacement for iptables
ufwSimplified firewall management (Ubuntu)
systemctlStart, stop, and check network services
